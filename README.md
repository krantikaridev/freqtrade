# freqtrade

Main trading system built on **Freqtrade** using a **Hybrid Sleeve** architecture.

### Objective
Build a scalable and maintainable trading system that can eventually generate meaningful income with reasonable operational effort.

### Core Principles
- Risk management first
- Quality over quantity
- Dynamic risk allocation between sleeves
- Strong focus on high-quality signals (especially from X)
- Iterative development

### Current Focus
- Designing and implementing a Hybrid Sleeve trading system
- Building strong risk management
- Setting up an efficient development workflow using GitHub Copilot + Grok

### Development Workflow
- High-level direction and tasks are maintained in this repository
- GitHub Copilot handles implementation
- Grok is used for architecture and strategy decisions

### Status
Early stage — setting up foundation and design.

**Note**: This is the primary trading system. `nanoclaw` is maintained separately as a tactical/on-chain experiment.
