# Roadmap - freqtrade

## Vision
Build a scalable Hybrid Sleeve trading system using Freqtrade that can serve as a meaningful income source.

## High-Level Phases

### Phase 1: Foundation (Current)
- Set up proper project structure and development workflow
- Define Hybrid Sleeve architecture
- Create strong risk management framework
- Establish prompts + Copilot + Grok workflow

### Phase 2: Core System
- Implement base Freqtrade configuration and strategy framework
- Build first sleeve(s) with clear edge
- Add proper logging, monitoring, and risk controls
- Backtesting and validation process

### Phase 3: Hybrid Sleeves
- Design and implement multiple sleeves with different characteristics
- Allocation logic between sleeves
- Dynamic risk adjustment between sleeves

### Phase 4: Automation & Scaling
- Deployment automation
- Monitoring and alerting
- Performance tracking and reporting
- Gradual capital increase

### Phase 5: Optimization
- Strategy improvement based on real performance
- Portfolio-level risk management
- Potential integration with other systems

## Success Criteria
- Consistent positive expectancy across sleeves
- Reasonable drawdown control
- Maintainable codebase with low operational burden
- Clear path to scale capital
